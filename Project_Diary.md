               PROJECT DIARY: Kg's KOTA DIGITAL MENU
======================================================================

1. THE GOAL (Why are we building this?)
----------------------------------------------------------------------
To build a lean, mobile-first single-page digital menu for the local 
kota spot across the street. The system will store orders in device 
memory (state) and compile a clean summary to text straight to the 
owner's WhatsApp, eliminating loose text message chaos.

2. SYSTEM BLUEPRINT OVERVIEW
----------------------------------------------------------------------
*   Frontend (The View): Semantic HTML5 and CSS Flexbox layout.
*   Backend (The Logic): Upgraded vanilla JavaScript ShoppingCart class.
*   The Bridge (The Sync): DOM Event listeners connecting buttons to RAM.
